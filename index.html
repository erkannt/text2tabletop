text2tabletop

Generate pretty PDFs for your OnePageRules armies.

  1. Build an armylist using Army Forge
  2. Use Share as Text in the Army Forge menu
  3. Paste below
  4. Print this page
When printing make sure to set your margins to default.
This project's code lives on Github. Feel free to raise issues and suggest features.
The code is Public Domain. I am in no way affiliated with OnePageRules, just enjoy playing their games and wanted prettier army list printouts.
